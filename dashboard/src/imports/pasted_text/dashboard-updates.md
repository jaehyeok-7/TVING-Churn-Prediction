[수정 1] 이탈 위험 분석 페이지 — KPI 카드 2개 수치 분리
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

현재 "Predicted Churn Users"와 "High Risk Users"가
둘 다 11,527명으로 동일한데, 아래 기준으로 다른 수치로 분리해줘.

① High Risk Users 카드
   → 수치: 11,527명 (유지)
   → 기준 라벨: "현재 churn_score ≥ 0.7 기준"
   → 부제: "현재 고위험군 사용자 수"
   → 컬러: #EF4444
   → 증감 배지: "+1,204명 ▲ 전주 대비"

② Predicted Churn Users 카드
   → 수치: 9,847명 (High Risk의 약 85%로 변경)
   → 기준 라벨: "30일 내 실제 이탈 예측 (신뢰구간 ±2.1%)"
   → 부제: "ML 모델 최종 이탈 예측 인원"
   → 컬러: #FF153C
   → 카드 하단에 작은 회색 텍스트 추가:
     "※ High Risk 중 재참여 가능 인원(1,680명) 제외한 순수 이탈 예측값"

③ Predicted Churn Rate 카드
   → 수치: (9,847 / 전체 유저) 기준으로 재계산 → 약 24.6%로 변경
   → 기준 라벨: "예측 이탈률 (churn_score 모델 기반)"

[수정 2] 개입 전략 페이지 — 중복 차트 교체
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

개입 전략 페이지의 "Churn Score Distribution" 히스토그램을
아래 "개입 전/후 Risk Band 변화 예측" 차트로 완전히 교체해줘.

[교체할 차트 스펙]
차트 제목: "개입 전략 적용 후 Risk Band 변화 예측"
부제: "캠페인 적용 시 30일 후 예측 분포 변화"
우측 상단 뱃지: "Simulation" (파란색 #3B82F6)

차트 타입: 나란히 배치된 묶음 막대 차트 (Grouped Bar Chart)
X축: 세그먼트 3개 — "High Risk" / "Mid Risk" / "Low Risk"
Y축: 유저 비율 (%)
막대 2개씩:
  - 빨간 막대: 개입 전 (Before)
  - 초록 막대: 개입 후 예측 (After)

데이터 (고정값):
const interventionData = [
  { segment: "High Risk", before: 30, after: 18 },
  { segment: "Mid Risk",  before: 40, after: 35 },
  { segment: "Low Risk",  before: 30, after: 47 },
]

각 막대 위에 변화량 표시:
  - High Risk: "-12%p ▼" (빨간색)
  - Mid Risk:  "-5%p ▼"  (주황색)
  - Low Risk:  "+17%p ▲" (초록색)

차트 하단에 인사이트 텍스트 박스 (배경 #1e2a1e, 초록 좌측 테두리):
"💡 캠페인 적용 시 High Risk 유저 중 약 40%(4,611명)가
    Mid/Low Risk로 전환될 것으로 예측됩니다."

범례: 🔴 개입 전 (Before)  🟢 개입 후 예측 (After)
